IDE: WinAVR
Команды для работы с терминалом

Символы вводятся в режиме ACSII, завершается нажатием клавиши Enter

1. Управление выводами порта. Доступны следующие выводы МК:

PORTB - пин 0, 1, 2, 6, 7
PORTC - пин 0-7

формат команды управления состоянием выводов: port x y z
x: b или c
y: номер вывода
z: 0 или 1

пример: port c 6 0 - выставить низкий уровень на выводе PC6
	port b 2 1 - выставить высокий уровень на выводе PB2

2. Управление генераторами

timer mode1 - включение генератора 1, 10, 100 Гц, задействованы выводы PC0, PC1, PC2 соответственно
timer mode2 - включение ШИМ-генератора, управление скважностью символами + или -
timer stop - остановка генератора

3. Отправка байта данных по SPI 

spi s 0x**

пример: spi s 0xaf (строчные буквы)
	spi s 0x56

4. Неблокирующая отправка буфера данных по SPI

spi s buf
